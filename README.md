# 🎯 Prompt Improve — AI 提示词写作教练

> 把你的 AI 从"许愿机"变成"专业医生"的6步提示词写作法。

一个面向 **OpenClaw** 用户的提示词写作技能（Skill），通过6步结构化引导，帮你写出高质量的 AI 提示词。

不需要背任何 Prompt 模板——**跟着对话走就行**。

---

## 📦 安装

### 前置条件

- [OpenClaw](https://openclaw.ai) 已安装并运行
- OpenClaw 版本 ≥ 1.0

### 安装步骤

```bash
# 1. 进入 OpenClaw 的 skills 目录
cd ~/.openclaw/workspace/skills/

# 2. 克隆项目
git clone https://github.com/A1kiii/prompt-improve-skill.git prompt-improve

# 3. 重启 OpenClaw，技能自动加载
openclaw restart
```

### 验证安装

在对话中发送以下任意触发词即可：

- 「帮我优化一下这个提示词」
- 「怎么写prompt」
- 「提示词教练」
- 「improve the prompt」

## 🧠 技能说明

### 核心原理：把 AI 当医生，不是许愿机

很多人把 AI 当许愿机，扔一句话就想要完美结果。

正确的心态是把它当**专业医生**——你描述越清楚，它诊断越准确。这个技能就是帮你养成这个习惯。

### 6步结构化引导

技能会按照以下顺序，**一次只问你一个问题**，等回答后再问下一个：

| 步骤 | 名称 | 问什么 |
|------|------|--------|
| 1️⃣ | **角色锚定** | 你希望 AI 扮演什么身份/角色？ |
| 2️⃣ | **语境铺陈** | 任务背景是什么？你在什么场景下用？ |
| 3️⃣ | **动作指令** | 具体要 AI 做什么？写/分析/翻译/策划？ |
| 4️⃣ | **边界控制** | 不要什么？字数限制？风格禁忌？ |
| 5️⃣ | **交付形式** | 想要什么输出格式？推荐 Markdown。 |
| 6️⃣ | **校验产出** | 帮你检查完整度，输出优化后的提示词。 |

> **核心口诀：** 角色、背景、指令、边界、格式——五个填空搞定 80% 的场景。

### 技能规则

- ✅ **一次只问一个问题**，不一次性扔出去信息过载
- ✅ 用户可以随时打断、跳到任意步骤、或回头修改上一步
- ✅ 如果你在回答某一步时已经包含了后续步骤的信息，技能会自动跳过已覆盖的问题
- ✅ 说「好了」「可以了」「就这样」直接跳到第6步校验产出
- ✅ 保持亲切但专业的教练语气

## 📖 使用教程

### 基础用法

最直接的方式：直接找教练帮你写提示词。

```
你：帮我写个提示词，我想让AI帮我列减肥计划

教练：好的！我们先从第一步开始——

🎭 角色锚定
你希望 AI 以什么身份来回答？
比如：专业的私人教练、资深营养师、还是两者结合？
```

然后你就回答，教练会继续问下一个问题。全程像聊天一样自然。

### 进阶用法

#### 优化已有提示词

直接把你的提示词丢过来：

```
你：帮我优化这个prompt——
"写一篇关于AI的文章"

教练：好的，我来帮您优化。先问第一个问题——
🎭 你希望 AI 以什么身份来写这篇文章？
比如：资深AI研究员、科技记者、还是……
```

#### 复制提示词给别人用

第6步校验产出后，教练会输出一个完整的 Markdown 格式提示词，可以直接复制到任何 AI 工具（ChatGPT、Claude、DeepSeek、Kimi……）中使用。

#### 复杂任务拆解

如果任务很复杂（比如写商业计划书），在第3步告诉教练，教练会提供拆解方案：

```
你：我想写一份完整的商业计划书

教练：这个任务比较复杂，我可以帮你拆成几步完成：
1. 先写项目概述和商业模式
2. 再分析市场和竞争
3. 然后做运营规划和财务预测
4. 最后整理风险和对策
你想从哪一步开始？还是我先帮你写出完整的提示词？
```

### 实用技巧

#### 给例子最有效

如果你描述不清楚想要什么感觉，告诉教练，他会引导你给例子。

**两种给例子的方式：**
- **正面实例**：给一篇你喜欢的范文
- **反面实例**：给一个你讨厌的样例

> 语言描述不清楚的感觉，两个例子就能让 AI 秒懂。

#### 角色越具体越好

角色锚定时，越具体越有效。可以指定出名的真实人物：

| 场景 | 有效锚定 | 弱锚定 |
|------|---------|--------|
| 健身规划 | "你是帕梅拉，我的私人教练" | "你是一个健身教练" |
| 情感建议 | "你是一个谈过一万次恋爱的情感达人" | "给我一些感情建议" |
| 志愿填报 | "你是张雪峰，中国高考志愿填报专家" | "帮我填志愿" |

### 完整使用示例

现实中的一次典型对话：

```
你：帮我写个提示词，我要让AI帮我写小红书文案

🤖 教练：好的！我们来一步步构建。

🎭 第一步：角色锚定
你希望 AI 扮演什么角色来写这个文案？
比如：资深小红书博主、美妆达人、生活方式编辑？

你：资深小红书博主

🤖 教练：好的。

📖 第二步：语境铺陈
这个产品/内容是什么？你的目标受众是谁？
是推广某个具体产品，还是分享生活经验？

你：推广一款平价面膜，受众是大学生和刚工作的女生

🤖 教练：明白了。

⚡ 第三步：动作指令
你具体要 AI 做什么？
是写一篇完整的笔记文案？还是起标题、写正文、还是写tag？

你：写完整的笔记，包括标题、正文、标签

🤖 教练：好的。

🚫 第四步：边界控制
有什么需要避免的吗？
比如：不要太营销感、不要太长、不要用夸张的修饰词？

你：不要太营销感，要像真实分享。不要超过500字。

🤖 教练：好的。

📋 第五步：交付形式
想要什么输出格式？推荐 Markdown。
还是你希望直接给小红书可以直接粘贴的纯文本？

你：纯文本就好，方便直接复制。

🤖 教练：好的，我来整理一下。

📝 第六步：校验产出
我检查一下…… 角色✅ 背景✅ 指令✅ 边界✅ 格式✅
完整的提示词如下：

```markdown
你是一个资深小红书博主，擅长写真实、接地气的好物分享。

我正在推广一款平价面膜，面向大学生和刚毕业的职场女生。

请帮我写一篇小红书笔记，包括：
1. 吸引人的标题（不超过20字）
2. 正文内容（500字以内）
3. 相关标签（5-8个）

要求：
- 语气真实自然，像朋友的真诚推荐，不要有营销感
- 不要使用夸张的修饰词和虚假使用效果
- 输出纯文本格式，方便直接复制粘贴
```

**质量评价：** 优秀。6个要素全部覆盖，边界清晰，格式明确。
**建议：** 如果使用后觉得语气不够真实，可以加个你心仪的博主名字做参考。效果会更好。
```

## ⚙️ 技术细节

### 触发机制

当用户消息匹配以下任何模式时，技能会被激活：

- 包含「prompt」或「提示词」
- 请求优化/改进/帮助写提示词
- 口语化表达如「怎么写prompt」「帮我写个提示词」
- 英文触发「improve the prompt」「prompt coach」

### 定制修改

想调整引导话术或增加步骤？直接编辑 `SKILL.md`：
- 修改各步骤的引导问题示例
- 调整校验清单的项目
- 添加新的框架模板

编辑后重启 OpenClaw 即可生效。
